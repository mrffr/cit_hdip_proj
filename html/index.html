Welcome

Welcome to my project!

This project contains information about incidents of crime reported in the city of San Francisco.
The data is provided by an Open Data API.

This information can be viewed as a number of user selected graphs.
It can also be viewed in the form of a series of maps, including a map of the latest 1,000 incidents in the database.

Please enjoy exploring the data and if you have any suggestions feel free to contact us.

The information on this website is updated every day, between 20:00 - 21:00 UTC.
