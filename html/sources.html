Sources

This project uses three open data APIs to gather information about incidents of crime that were reported in San Francisco.

Open data is the idea that data should be both free to use and publicly available, so that people may access, use, modify and redistribute the data.

Project APIs

The three APIs that I used are listed below.

San Francisco Police Department Incidents API

Police Department Incident Reports: 2018 to Present
This API returns information on incidents of crime that have been reported in San Francisco, it is the main source of data used in this project.

San Francisco Geographical APIs

These next two APIs were used to create the choropleth maps.

San Francisco Police Districts
This API was used to get the geographical boundaries of the San Francisco Police Districts.

San Francisco Neighbourhoods
This API was used to get the geographical boundaries of San Francisco neighbourhoods.
